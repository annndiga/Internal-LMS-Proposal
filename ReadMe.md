## LMS Frontend
This is the frontend for the Learning Management System (LMS), built with React.js and Tailwind CSS.

### Folder Structure
- **components/**: Reusable UI components
- **pages/**: Main application pages (Home, Dashboard, Courses, etc.)
- **context/**: Global state management for authentication and courses
- **services/**: API service handlers
- **routes/**: Handles app routing
- **assets/**: Static images, icons, and styles

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/lms-frontend.git
   ```
2. Navigate to the project directory:
   ```sh
   cd lms-frontend
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```

### Technologies Used
- React.js
- Tailwind CSS
- React Router
- Context API for state management
